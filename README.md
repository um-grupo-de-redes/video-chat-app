# Instalação e Execução

## Servidor

Requisitos:
* Docker instalado.
* Docker Compose instalado.

#### Subir servidor localmente

```
docker compose -f docker-compose-local.yml up
```

#### Subir servidor na produção (com SSL)

```
docker compose up
```
